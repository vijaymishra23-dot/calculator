# Calculator

A simple browser-based calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage and sign toggle
- Division by zero protection
- Clean dark UI

## Usage

Open `index.html` in any browser — no install or build step needed.

## Project structure

```
calculator/
├── index.html      # Structure and button layout
├── style.css       # Dark theme styling
└── calculator.js   # All calculator logic
```
